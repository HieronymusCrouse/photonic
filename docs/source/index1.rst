Photonic User Guide
=======================

Welcome to the Photonic User Guide. These pages lists general usage guidelines on the Photonic Web User Interface.

Introduction
------------
When visiting the URL of the Photonic installation in a web browser, one is presented with a login screen.
The default username is *root*, and the default password is *password*.

.. image:: /_static/img/login.png

An example of the login screen to Photonic.

The third field can be used to log into a specific :ref:`domain <domains>`, and is optional. If left blank, one will log into the *default* domain.
